Project Stardust

Users must meet a set of requirements that make sure the habitat is not too heavy, too expensive, and to make sure it contains crucial needs like energy generation, communication, waste management, filtration, etc. This project is beneficial compared to other existing options because, unlike many tools for these purposes, our software is easy to use for all users. Most software that is used for 3D design/building uses complex architecture or requires high-level knowledge, but our software is simple, while still being able to bring in complexity and structure. We aim to make knowledge and design usable by all people, regardless of prior knowledge. Our team considered how to balance simplicity and realistic approaches, coming up with a creative balance to best optimize the project for ALL users.
